## Firmware
I used [KMK](https://github.com/KMKfw/kmk_firmware/tree/main). Right now the keys are set to numbers 1-9 and the rotary encoder changes volume, but these are placeholders. I'll update the firmware and take advantage of the OLED screen once I have the case and PCB assembled.

## BOM

Here's all the parts needed:
- 8 x Cherry MX Switches
- 8 x DSA Keycaps
- 3 x M3x16mm screws
- 9 x 1N4148 Diodes
- 12 x SK6812 MINI-E LEDs
- 1 x 0.91" 128x32 OLED Display
- 1 x EC11 Rotary Encoder
- 1 x XIAO RP2040
- 1 x Case (2 x 3D printed parts)
